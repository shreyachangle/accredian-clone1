
# 🚀 Accredian Clone - EdTech Landing Page

A modern **EdTech landing page clone** built using **Next.js, React, and Tailwind CSS** with a clean production-style UI.

## ✨ Features
- 🎯 Modern SaaS-style landing page
- 📚 Programs section with icons, price, and badges
- 🏢 Enterprise corporate training page
- 📞 Contact page with form UI
- 🔐 Login page UI
- 🎨 Fully responsive design
- ⚡ Fast performance with Next.js App Router
- 🧩 Component-based architecture

---

## 🛠️ Tech Stack
- Next.js 14
- React.js
- Tailwind CSS
- Lucide React Icons

---

## 📂 Pages
- `/` → Home (Hero + Sections)
- `/programs` → Course listings
- `/enterprise` → Corporate training
- `/contact` → Contact form
- `/login` → Login UI
